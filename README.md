# University Conferences

A project to create a proper database of NCAA DI, DII, DIII, and NAIA conference membership over time.

Primary goal is to leverage Wikipedia pages as a corpus of information against which retrieval-augmented generation (RAG) can be used to generate timelines of conference membership for universities.

Given that NCAA membership alone is >1000 members and I cannot identify a database or file online with this information already, my goal is to create one for all to use.
